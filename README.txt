Kleines Plugin für GIMP, das es erlaubt, das gesamte sichtbare Bild als DDS zu exportieren. Das DDS-Plugin für GIMP kann dies nur für die gerade aktive Ebene.

Zur Installation wird die Datei `export-visible-dds.py` in einen der Plugin-Orgdner kopiert. Der Pfad zum Plugin-Ordner steht in den GIMP-Einstellungen unter Ordner → Plugins. Das DDS-Plugin für GIMP (http://code.google.com/p/gimp-dds/) muss ebenfalls installiert sein.

Der Export ist erreichbar unter Datei → Sichtbares als DDS exportieren.
